# Lithhp Agent Instructions

This document provides instructions for agents working on the lamedh Lisp interpreter.

## Building

To build the project, run the following command:

```bash
cargo build
```

## Running

To run the REPL, use the following command:

```bash
cargo run
```

## Testing

To run the test suite, which includes unit and integration tests, use the following command:

```bash
cargo test
```

## Code Style

Please run `cargo clippy` to check for linting issues and `cargo fmt` to format the code before submitting.
