# Lithhp
A Lisp 1.5 implementation.

It will:

- read
- eval
- print

it supports:

math operations + - / *
string operations concat ++, index
strings are arrays. 
list operations car cdr cons

it cam define functions (defun), and has lambdas, if, let.

it should support fexprs with a defexpr. it should have quote, quasiquote, and unquote ('`,). 

syntactically it is a classic s-expression system.
